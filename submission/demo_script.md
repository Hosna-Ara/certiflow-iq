# CertiFlow IQ Demo Script

## Opening
CertiFlow IQ is a multi-agent certification readiness system for enterprise teams. It helps learners prepare for role-based certifications and helps managers identify team readiness risks.

## Problem
In many organisations, certification planning is difficult because learners have different roles, study progress, workloads, and readiness levels. Managers also need a clear view of who is ready, who needs review, and who is at risk.

## Solution
CertiFlow IQ uses specialised agents to reason over synthetic learner data, workload signals, certification requirements, and approved learning documents.

## Demo Flow
1. Start on the Manager Insights tab.
2. Show total learners, ready learners, learners needing review, and at-risk learners.
3. Explain that the system gives a team-level readiness view.
4. Move to Learner Readiness.
5. Select learner L-1001 or L-1002.
6. Show how the Learning Path Agent maps role to certification skills.
7. Show how the Study Plan Agent creates a workload-aware plan.
8. Show how the Engagement Agent recommends realistic reminders.
9. Show how the Assessment Agent checks readiness using score and study-hour thresholds.
10. Show grounded practice questions with source references.
11. Show the Safety / Verifier Agent.
12. Move to Agent Trace and show the execution order.

## Microsoft Foundry and IQ Explanation
A Microsoft Foundry resource was created successfully in Southeast Asia using Azure for Students. Model deployment was blocked by quota, so the current version runs locally while keeping the architecture Foundry-ready.

Foundry IQ is represented through synthetic knowledge documents prepared for grounding. Fabric IQ is represented through the semantic model connecting roles, certifications, skills, hours, and readiness thresholds. Work IQ is represented through workload signals such as meeting hours, focus hours, and preferred learning slots.

## Closing
CertiFlow IQ demonstrates multi-agent reasoning, role-based planning, grounded assessment, workload-aware adaptation, manager insights, and responsible synthetic-data use for enterprise certification readiness.
