# Agent Responsibilities

## 1. Orchestrator Agent
Receives the user request, identifies whether the request is from a learner or manager, and coordinates the correct specialist agents.

## 2. Learning Path Curator Agent
Maps learner roles and certification targets to relevant skills, study areas, and approved learning materials.

## 3. Study Plan Generator Agent
Creates a practical study schedule using certification requirements, recommended hours, learner progress, and workload capacity.

## 4. Engagement Agent
Uses synthetic workload signals such as meeting hours, focus hours, and preferred learning slot to suggest realistic reminder and study timing.

## 5. Assessment Agent
Evaluates learner readiness using practice score, study hours, readiness threshold, and grounded knowledge sources.

## 6. Manager Insights Agent
Summarises team-level certification readiness, risk areas, capacity concerns, and recommended manager actions.

## 7. Safety / Verifier Agent
Checks that outputs are based on synthetic data, avoid personal data exposure, and include safe recommendations.
