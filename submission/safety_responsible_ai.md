# Safety and Responsible AI

## Synthetic Data Only

CertiFlow IQ uses synthetic learner, workload, certification, and team data only.

The project does not use:
- Real employee data
- Customer data
- Personal emails
- Personally identifiable information
- Credentials
- Confidential organisational documents

## Data Protection

All demo identifiers are fictional, such as:
- L-1001
- EMP-001
- TEAM-A

The `.env` file is excluded from GitHub using `.gitignore`.

## Guardrails

The Safety / Verifier Agent checks that:
- Outputs are based on synthetic data
- No real personal information is exposed
- Readiness recommendations are advisory only
- Assessment outputs do not claim official exam results
- Manager insights are shown at team level where possible

## Human Oversight

CertiFlow IQ supports decision-making but does not replace managers, trainers, or certification advisers. Final exam decisions should remain under human review.

## Transparency

The application clearly informs users that they are interacting with an AI-assisted demo system using synthetic data.
