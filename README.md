# CertiFlow IQ

CertiFlow IQ is a multi-agent certification readiness system for enterprise teams, built for the Reasoning Agents with Microsoft Foundry challenge.

## Problem

Organisations often struggle to manage employee certification programmes across different roles, workloads, skill gaps, and readiness levels. CertiFlow IQ helps learners and managers create role-based study plans, assess readiness, and identify team-level certification risks.

## Core Users

- Learners preparing for role-based certifications
- Managers monitoring team certification readiness

## Multi-Agent System

- Orchestrator Agent
- Learning Path Curator Agent
- Study Plan Generator Agent
- Assessment Agent
- Manager Insights Agent
- Safety / Verifier Agent

## Microsoft IQ Usage

- Foundry IQ: grounded certification content and cited practice questions
- Fabric IQ: semantic model for role, certification, skills, thresholds, and readiness
- Work IQ: synthetic workload signals for capacity-aware study planning

## Data Notice

All learner, workload, certification, and team data used in this project is synthetic and for demonstration only. No real employee data, customer data, credentials, or personally identifiable information are used.
